Lab1. Simplex Method

How to use:

In your shell run

```Shell
>>make main(or just make)
>>main input output
```
E.g

```Shell
>>make
>>main simplex.in simplex.out
>>less simplex.out
```

In simplex.out there will be a resulting simplex table, where the lowest-left element is minimal or maximal value of target function
